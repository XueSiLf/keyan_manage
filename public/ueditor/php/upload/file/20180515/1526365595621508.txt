《15》sass安装步骤：
     1.下载node.js,      查看版本号   node -v；
     2.安装cnpm  打开命令行cmd  cd到用户 执行npm install cnpm；
     3.npm install -g cnpm     查看版本号   cnpm -v;
     4.cnpm安装好之后安装sass   cnpm install -g node-sass;
     5.检测是否安装成功    node-sass -v;
     6.node-sass -w index.scss index.css;
